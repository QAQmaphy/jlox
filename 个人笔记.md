```
  Token.java中定义了具体的数据类,这个类中包含了一个数据类的类型、词素、字面量值、行号。词素就是原始的文本，字面量值就是在运行时，真正的值是多少，词素永远是字符串
  比如数字1，词素就是字符串1，但是运行时就是一个整数1（在该项目中整数使用浮点数表示）
  类型用于之后的逻辑判断，
  词素用于报错信息、变量绑定、变量检索
  行号用于错误定位

  在expr.java和stmt.java中,存放着表达式和语句的定义
  表达式的语义是计算出一个值,返回一个观察者对象, 表达式中是运算符和数据的集合,通过语法解析器将token流翻译成一个个的表达式和语句,并将这个表达式和语句节点按照优先级关系构造成一颗树.
  语句的语义是执行副作用,这里的副作用指的就是赋值,输出等作用,往往没有返回值,
  因为表达式和语句都有返回观察者对象的方法accept,所以说可以通过递归调用该节点的观察者对应方法的方式来进行对自己的解释.比如一个二元表达式(如1 + 2) ,通过观察之后,返回3,再由var语句将其赋值给一个标识符.
  目前来说一共有三种语句,有表达式语句,赋值语句和打印语句
  语句节点会调用观察者的与自己对应的visit方法,观察者需要根据节点的要求来实现对应的visit方法,在使用的时候,顺序是观察者将自己的方法"都"给到当前的节点(实际上就是将自己当做参数传给accept),当前节点再去调用该观察者观察自己的方法.
```

# 词法分析

``` 
对于单字符的符号，直接将其识别成token

二字符的符号要前瞻两位（包括注释）

遇到一个双引号的时候，进入扫描字符串的模式，不断地在一个while循环理进行前瞻，然后消费一个字符，直到遇到另一个双引号停止循环，如果一直到了文件尾部还没有遇到双引号，说明双引号没有闭合，跳出错误。如果正常退出，消费掉这个双引号，使用子字符串操作将该字符串提取出来，

数字，要进行小数点的判断：先不断消费数字，停下后，判断之后的数字是不是一个小数点，是的话就消费掉，然后进行循环消费数字。

标识符、关键字：在发现扫描到一个英文字母的时候，不断循环消费后面的字母，然后将这次循环获取到的字符串提取出来，与一个关键词字典进行对照，如果是不是关键字，那就将类型定义为标识符
```
